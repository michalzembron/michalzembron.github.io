Endless Dungeons

'Endless Dungeons' - A 3D Hack and Slash game made in Unity engine

Objective of the work

The aim of the work was to create a game and the elements necessary for its operation, such as 3D graphics, textures, animations and scripts.
The scope of the work included:
- familiarization with the following programs:
    - Unity 3D - an environment used to create games,
    - Blender - used for 3D modeling, object texturing and creation
      animation,
    - GIMP - a program for editing 2D graphics, in this case creating textures,
    - Microsoft Visual Studio - a development environment that allows you to create scripts in C # for the needs of the game,
- gathering and getting acquainted with the material concerning the above programs, as well as the general aspects of creating computer games,
- creation of:
    - lists of elements necessary for "Hack and Slash" games,
    - 3D models, such as: characters with the creation of their skeleton and animations, buildings, plants and other decorative elements,
    - appropriate scripts in C#,
    - textures for the models and the terrain,
    - testing and removing bugs.
